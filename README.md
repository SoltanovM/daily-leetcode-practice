# Daily LeetCode Practice

This repository contains my daily practice of data structures and algorithms using LeetCode, using mainly Python, shell scripts, and SQL.

## Goals

- Improve problem-solving and algorithmic thinking
- Strengthen understanding of core data structures
- Write clean, readable, and efficient code
- Analyze time and space complexity
- Maintain consistency through daily practice

---

## Structure

Problems are organized into the following folders:

```md
blind-75/
sql-questions/
shell-script/
miscellaneous/
```

- `blind-75/` contains solutions for the classic Blind 75 problem set.
- `sql-questions/` includes LeetCode SQL practice problems and query solutions.
- `shell-script/` contains shell scripting challenges and command-line solutions.
- `miscellaneous/` is for any additional problems that do not fit neatly into the other categories.

## Why this repository?

This repo reflects:

- Consistency in learning
- Structured problem-solving approach
- Focus on writing clean and maintainable code
- Continuous improvement mindset
